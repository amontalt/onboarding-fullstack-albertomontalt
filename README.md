# Fullstack Task Manager - Alberto Montalt

Este repositorio contiene mi proyecto final de Onboarding Fullstack, diseñado como base para mi TFG y Portfolio profesional. Es una aplicación robusta de gestión de tareas con arquitectura desacoplada.

## 🛠️ Tecnologías Core
- **Frontend:** Angular 17 con arquitectura de módulos y componentes reutilizables.
- **Backend:** Symfony 7 (PHP) funcionando como API robusta.
- **Autenticación:** Seguridad implementada con JWT (JSON Web Tokens).
- **Infraestructura:** Despliegue orquestado con Docker y Docker Compose.

## 📁 Estructura del Proyecto
- `angular-frontend/`: Aplicación cliente Single Page Application (SPA).
- `symfony-backend/`: API RESTful con persistencia en base de datos.
- `docs/`: Documentación técnica y presentación personal.

## 🚀 Instalación rápida
1. Clonar el repositorio.
2. Ejecutar `docker-compose up -d` en la raíz.
3. Acceder a la app en `http://localhost:4200`.
